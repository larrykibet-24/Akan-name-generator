Akan Name Generator
Description

Discover your traditional Ghanaian identity with this simple web tool. In Akan culture, children are named based on the day of the week they were born. Simply enter your birthdate and gender to reveal your unique "day name" and its cultural significance.

Live link : https://github.com/larrykibet-24/Akan-name-generator

Author
Larry Kibet

Setup Instructions
Clone this repository
Open index.html in your web browser
Enter your birth date and select your gender
Click submit to see your Akan name
BDD (Behavior Driven Development)
Input: Birth date and gender

Output: Akan name based on day of birth and gender
Input: Invalid date (day ≤ 0 or day > 31)

Output: Error message requesting valid date
Input: Invalid month (month ≤ 0 or month > 12)

Output: Error message requesting valid month
